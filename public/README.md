# Public Assets Directory

Please upload your transparent logo image here and name it `logo.png`.

### How to upload:
1. In the file explorer on the left of Google AI Studio, locate or create the **public** folder.
2. Select or drag-and-drop your image into this folder.
3. Rename the uploaded image to `logo.png`.

The application's `<Logo />` component is configured to automatically load `/logo.png` from this directory. If it's missing or hasn't been uploaded yet, it will dynamically fall back to the high-fidelity SVG/CSS metallic styled logo.
